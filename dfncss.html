BELAJAR CODING WEB PROGRAMING

  • CSS

CSS sebuah bahasa sederhana yang digunakan untuk menambahkan gaya/styling (misalnya, font, warna, spasi) ke sebuah halaman website. Jika diibaratkan, HTML merupakan sebuah kerangka, CSS ini bertindak sebagai kulit/penutup dari kerangka tersebut. Adapun fungsi CSS bisa mengatur mulai dari font, warna tulisan dan latar belakang pada website tersebut.
